# *Arena*
Arena es un juego multijugador competitivo, que sera utilizado como proyecto personal para aprender Game development.
<hr/>
### Items Implementados
* Jugador
  * Controlador de Movimiento
  * Controlador de Arma
* Enemigo
  * Pathfinding
* Gameplay
  * Sistema de daño
  * Ricochet de las balas
